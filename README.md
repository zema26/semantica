Semantica

I'm designing new programming language with very creative syntax. I want to unify all language syntax with right arrow symbol " > " .  It means that assignment operator:

' a + b > a '

function declaration:

' sub| int a, int b > Euclid : int ... '

function call:

' a, b > Euclid '

and I/O :

' in > a,b > Euclid > out '

looks the same under this clear unified pattern. That makes writing the code much  easier and more effective. Especially for AI

You can look at language description, specification and transpiler to TypeScript at 
https://semantica.replit.app
